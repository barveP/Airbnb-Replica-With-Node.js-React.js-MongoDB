# Airbnb-Replica-With-Node.js-React.js-MongoDB

Fullstack Airbnb replica with Node.js, MongoDB backend and React.js frontend:

Features for Hosts/Owners:
> Post property listings, 
> Manage properties,
> Manage bookings

Features for Travellers:
> Search homes,
> Make bookings,
> Manage bookings
